Cytoscape ESP – A simple search method for the exploration of complex biological networks.


This folder contains the work done by August 20th, 2007. However, this project is still evolving!

The latest source code can be found on (and downloaded from) Cytoscape’s Subversion server.
http://www.cytoscape.org/cgi-bin/moin.cgi/SVN_Access

You can download a recent jar file of the plugin from
http://www.agri.huji.ac.il/~moshelion/lab_members/maital

To test the plugin, install Cytoscape and place the jar file under the plugins folder:
http://www.cytoscape.org/download_list.php
